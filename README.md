# dotnet-selenium

`dotnet add package Selenium.WebDriver`
`dotnet add package MSTest.TestFramework`
`dotnet add package MSTest.TestAdapter`
`dotnet add package Microsoft.NET.Test.Sdk`

`dotnet build`
`dotnet run`

`cd test/`
`dotnet test -s .runsettings`
